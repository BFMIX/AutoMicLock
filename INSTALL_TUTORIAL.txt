How to install and run AutoMicLock if blocked by macOS

1. Drag the AutoMicLock.app icon to the Applications folder (as shown above).

2. After copying, try to launch AutoMicLock from your Applications folder or from the Menu Bar.

3. If you see a message that “AutoMicLock cannot be opened because the developer cannot be verified”:
   a. Click “OK” to close the warning.
   b. Open System Settings (or System Preferences) → Privacy & Security.
   c. Under the “Security” section, you will see a message indicating that “AutoMicLock was blocked from use because it is not from an identified developer”.
   d. Click the “Open Anyway” button next to that message.
   e. A confirmation dialog will appear; click “Open” to launch the application.

4. Once opened, the app will run in your menu bar and you can enable the microphone lock from its interface.

Note: This warning appears because the app is not signed with an Apple Developer ID. Signing requires a paid Apple Developer Program membership. The app is open‑source and does not contain any malicious code. You can inspect the source code on GitHub at https://github.com/BFMIX/AutoMicLock.
