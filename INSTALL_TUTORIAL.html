Comment autoriser l'ouverture d'AutoMicLock

Étape 1 : Le message de blocage

Lorsque vous lancez l'application pour la première fois, Apple bloque l'ouverture car elle provient d'un "développeur non identifié". Cliquez simplement sur Terminé (Done) ou OK.

Étape 2 : Réglages Système

Ouvrez les Réglages Système (System Settings) de votre Mac, puis allez dans l'onglet Confidentialité et sécurité (Privacy & Security). Descendez dans la page, et vous verrez un message concernant AutoMicLock. Cliquez sur le bouton Ouvrir quand même (Open Anyway).

Étape 3 : Confirmer l'ouverture

Une dernière fenêtre de confirmation s'affiche. Entrez le mot de passe de votre session (si demandé) ou cliquez directement sur le bouton Ouvrir (Open).

Note : Cette manipulation n'est nécessaire que lors du tout premier lancement. Par la suite, l'application s'ouvrira normalement.
