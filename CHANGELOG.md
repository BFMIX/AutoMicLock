# Changelog

All notable changes to AutoMicLock will be documented in this file.

## [v1.1.1] - 2026-05-16

### 🚀 Improvements
* **Modern Footer UI:** Redesigned the footer with stylish buttons and a centered version indicator.
* **Dynamic Icons:** Finalized Light/Dark mode application icons.
* **Security & Reliability:** Integrated real EdDSA signing keys and added a debounce delay for stable audio switching.
* **Accessibility:** Full VoiceOver support for all UI elements.

### 🐛 Bug Fixes
* Resolved update loop and signature validation errors.
* Fixed DMG background scaling and layout issues.
