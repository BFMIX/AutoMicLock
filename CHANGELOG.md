# Changelog

All notable changes to AutoMicLock will be documented in this file.

## [v1.1] - 2026-05-16

### 🚀 New Features & Improvements
* **Dynamic Menu Bar Icon:** The menu bar icon changes shape and color based on the microphone lock status.
* **Smart Device Switching:** Added a short delay mechanism to prevent rapid audio switching loops.
* **Dynamic Appearance:** The application icon automatically adapts to your macOS system theme (Light or Dark mode).
* **Accessibility:** Added VoiceOver support to all UI elements to make the app more inclusive.

### 🐛 Bug Fixes
* Fixed an issue where the update checker would crash on the first launch.
* Fixed visual scaling issues in the installation window.
